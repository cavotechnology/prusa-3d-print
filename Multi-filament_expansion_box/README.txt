# Summary

This is the second version, but first to be published.
Container for collecting the retracted filament in a multi-filament system like Prusa MMU1.0/2.0 or Palette.

Dividers swing down to accommodate filament loading.
dividers can be substituted for 3mm thick anything-you-have-on-hand

Required parts:

1- 8mm shaft (same found in Prusa MMU 2.0 spool holders)
5- 608 skate bearings (same found in Prusa MMU2.0 spool holders
2- M3x 10mm screws
4- M3x 16mm screws
4- M3 hex nuts
4- wall mounting hardware of choice (if applicable)
5- PTFE 50mm long for top
5- PTFE (distance to MMU tubes) minus 40mm
1- Body
5- rollers
6- dividers (if applicable)
1- top PTFE holder
2- ptfe large
2- PTFE large joint

I use short tubes from MMU in my situation, using the 40mm as a way for me to manipulate the filament if needed.
